Genereate random posts for your site

== Description ==

This plugin generates random content for your site. You can:

- select post type, taxonomy, term
- chosse number to posts to create
- paragraph long, ul, ol, and so on
- you can add images (coloured block image) to check your templates. You can select a color or choose randomly. You can set width and height for the posts thumbnail
- if you have Woocommerce instaled, you can create random products, select range price, and so on.
- when you have the real data in your site, you can remove all the posts generated by the plugin pressing a simple button
== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/postem-ipsum` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Postem Ipsum menu itemn in the Wp Administrator screen to configure the plugin and create the posts.
